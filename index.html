Pre-Commands · Camera & Focus Camera
<<focus characterName>>
Sets the focus to a character. Required before skill rolls. Only one focus per node.
▶

C# Registration

dialogueRunner.AddCommandHandler( "focus", (string characterName) => { // Find and cache the focused character focusedCharacter = CharacterManager .FindByName(characterName); } );

Yarn Usage

// Focus Alaron before a skill roll <<focus Alaron>> <<roll Loremaster>>
<<cutToCamera referencePointNumber>>

Yarn Usage

<<trackon referencePointNumber>>
Linearly tracks (moves) the camera to a reference point. Always pair with <<speed>>.
▶

C# Registration

<<speed speedValue>>
Sets the speed for any subsequent linear camera movement. Use slow (30), medium (10), fast (5), or a raw integer in 60ths of a second.
▶

C# Registration

int currentSpeed = 10; // module-level dialogueRunner.AddCommandHandler( "speed", (string value) => { currentSpeed = value.ToLower() switch { "slow" => 30, "medium" => 10, "fast" => 5, _ => int.Parse(value) }; } );

Yarn Usage

<<speed slow>> // 30 frames <<speed medium>> // 10 frames <<speed fast>> // 5 frames <<speed 20>> // 20/60ths of a second
Variable Retrieval Commands Variables
Note: Variable retrieval commands are used inside if and switch nodes. if nodes only accept boolean results; switch nodes accept boolean or integer. In YarnSpinner, these are implemented as custom functions that return values.
checkgold()
Returns the total gold the party currently holds.
→ int▶

C# Registration

dialogueRunner.AddFunction( "checkgold", () => PartyManager.GetGold() );

Yarn Usage

<<if checkgold() >= 100>> You have enough gold. <<endif>>

Scene Setup Commands Game Flow
<<setup>>  /  <<stage stageID>>  /  <<start>>
Used in scripted dialogs. <<setup>> specifies the scene location, <<stage>> selects a stage template (1–8), and <<start>> begins playback after all initialization is complete.
void▶

C# Registration

dialogueRunner.AddCommandHandler( "setup", () => SceneManager.InitializeScene() ); dialogueRunner.AddCommandHandler( "stage", (int stageId) => SceneManager.SetStage(stageId) ); dialogueRunner.AddCommandHandler( "start", () => SceneManager.BeginPlayback() );

Yarn Usage

// Always: setup → stage → place → start <<setup>> <<stage 3>> <<focus Ehud>> <<place WP_Center>> <<start>>

These commands (cut, moveLinear, moveSine, position) predate the current camera system. They are superseded by <<cutToCamera>>, <<angleTo>>, <<sineTo>>, <<trackon>>, <<speed>>, and the POV variants. All new dialog should use only the Pre-Commands listed at the top of this reference.
